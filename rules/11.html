Web-O-Tronic

Like designing web page?! Like to play with CSS and HTML?! This is about the web-designing event of Avenir 2015. It will consist of 2 rounds.

A prelims round, and
A final round.

Rules and Regulations:

Team Size :
Maximum 2.

Prelims

There will be a 30 minutes test, having 30 Objective type questions testing the basic HTML and CSS skills of the participants.

The teams scoring above a certain cut-off will advance to the Finals.

Final

All the teams will be given a simple HTML/CSS template, a set of instructions and the basic resources required for designing a web page. Their task is to modify the given template as per the given problem statement, and show their own creativity as well, if possible.

Extra features carry additional credits.

Time provided to complete the task is 3 hours.The team which takes the minimum time to fulfill do their job wins.

The teams are not allowed to bring any CSS code of their own, they have to use from the CSS codes, that we will provide, and modify them.

Hardcopy resources can be used for help, no softcopies allowed, and also no online help will be provided.

The decision of the judges will be final and undisputable.

(Additional software like notepad++ or Dreamweaver will be provided)
